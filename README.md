# Music_Game
Викторина по песням, альбомам и интервью группы One Direction.  
Режимы: угадай песню по отрывку, альбом по обложке, участника по интервью.

## Требования  
- ОС: Windows 10/11  
- Компилятор: GCC (MinGW)  
- CMake 3.10+

## Сборка  
```bash
cmake -B build
cmake --build build
./build/music_game
